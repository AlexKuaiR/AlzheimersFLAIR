# AlzheimersFLAIR

This repository contains code for an Alzheimer’s disease imaging analysis project using ADNI data.

## Data availability

Raw and processed ADNI participant-level data are not included in this repository due to ADNI data-use restrictions. Approved users should obtain data directly through the official ADNI/LONI access process.

This repository includes code, documentation, and project structure only. Participant-level images, labels, processed arrays, and model artifacts are intentionally excluded.

## Repository contents

- Preprocessing code
- Model training/evaluation code
- Cleaned notebooks with outputs removed
- Documentation for reproducing the local workflow with approved ADNI access
